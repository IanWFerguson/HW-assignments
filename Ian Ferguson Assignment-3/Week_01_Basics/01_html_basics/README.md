## Lesson 01 - HTML Basics

### LEARNING OBJECTIVES

*	Apply HTML tags ```<html><head><!DOCTYPE html><body>``` to a web page and experiment with html tags.

*	Describe the DOM and draw a simple DOM tree.

*	Create and link an external style sheet.
